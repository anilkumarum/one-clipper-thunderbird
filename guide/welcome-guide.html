How to connect your oneNote

  1. After install extension, Open extension's popup window by clicking on extension's icon or keyboard shortcut (Alt+O).
  2. One Note connection dialog auto popup.
  3. If want to connect your Office 365/Work/School account then select Work/Office 365 option. Click on Connect with OneNote button.
    4. ℹ️ Info : You can add multiple One Note account

How to send message to One Note

  1. Open extension's popup window by clicking on extension's icon or keyboard shortcut (Alt+O).
  2. Click on to send message to One Note
